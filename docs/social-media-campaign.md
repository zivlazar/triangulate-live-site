# Triangulate Social Media Campaign

Draft date: 2026-04-23

This plan uses the Triangulate website as the reference point for a TikTok and Instagram presence. The website position is:

- Outdoor multiplayer game.
- "Run. Claim. Survive."
- Three players form a triangle in real space.
- Events, teams, and leaderboards are the main ways into the game.
- The tone is energetic, local, social, and grounded in real outdoor play.

The campaign should talk about events, places, formats, tactics, movement, and the feeling of the game. It should not talk about individual players unless there is explicit consent.

## Goals

Build a recognisable Triangulate presence on TikTok and Instagram that:

- Creates promotional content several times a day.
- Turns the public events list into a reliable content trigger.
- Uses nearby gaming, sports, student, fitness, and local-event conversations to attract relevant audiences.
- Builds repeatable hashtags, event tags, and post labels that make Triangulate easy to find.
- Can move from human-assisted posting to official API-based automation once accounts and permissions exist.

## Content Positioning

Core phrase:

```text
Three points. One triangle. Real-world play.
```

Supporting phrases:

- `Run. Claim. Survive.`
- `Outdoor multiplayer, played on a real map.`
- `A local event turns into a live triangle match.`
- `Bring mates. Spread out. Hold the shape.`
- `The game starts when the event starts.`

Voice:

- Short, visual, energetic, and local.
- Use plain English over gamer jargon.
- Make the game feel easy to understand but hard to master.
- Sound like a local event host, not a corporate app.

Do:

- Talk about the venue, format, timing, routes, space, weather, and energy.
- Explain how the triangle works.
- Promote event discovery and joining.
- Use city and neighbourhood names when relevant.
- Invite people to watch, join, or bring a group.

Do not:

- Name individual players, show private player stats, or discuss performance without explicit consent.
- Use player faces in generated or real content unless consent is documented.
- Reply to unrelated posts with generic ads.
- Claim affiliation with other events, clubs, schools, or sports groups unless confirmed.
- Use scraping, browser bots, or unofficial automation to mass-like or mass-comment.

## Daily Cadence

Start with this cadence for the first 30 days:

- TikTok: 2 to 3 posts per day.
- Instagram Reels: 1 to 2 posts per day.
- Instagram Stories: 3 to 6 frames per day on event days.
- Instagram feed/carousel: 3 posts per week.

Suggested UK posting windows:

- Morning: 08:00 to 09:00, event discovery and "what is Triangulate?"
- Lunch: 12:00 to 13:30, quick explainers and team/session prompts.
- After school/work: 16:00 to 18:30, event reminders and local hooks.
- Evening: 20:00 to 22:00, recaps, tactics, and tomorrow's event teaser.

Do not force volume when there are no relevant events. If the event list is quiet, switch to gameplay explainers, local spaces, and team-session content.

## Event Trigger System

The event list is the main trigger. The current app and website expose event fields such as title, plan note, scheduled time, status, meeting point name, parent venue, postcode, photo URL, latitude, longitude, registration count, team name, and team-member count.

For social content, use only event-level facts:

- Event title.
- Venue or neighbourhood.
- Scheduled day/time.
- Format or plan note.
- Public photo or generated venue-style graphic.
- "Upcoming", "starting soon", "live", or "recent" state.

Avoid:

- Creator nickname.
- Individual player names.
- Individual player stats.
- Faces or identifiable user-generated media unless consent exists.

Triggers:

- New approved/open event: create announcement draft for TikTok, Reel, Story, and static IG post.
- T minus 72 hours: explain the venue and format.
- T minus 24 hours: "tomorrow" reminder with a simple gameplay tip.
- T minus 3 hours: "starting soon" reminder.
- Event live: Story-only prompt, unless someone is manually monitoring.
- Event ended: recap the place and format, not the players.
- Event cancelled/rejected/pending: no public promotional post.

Trigger priority:

- Flagship or city event.
- Local event happening within 24 hours.
- Event with a strong plan note or distinctive venue.
- Away-day or team session.
- Recently completed event with a useful recap angle.

Example event-triggered caption:

```text
Victoria Park becomes a triangle field tonight.

Three points, one shape, and a live map that only works when the team moves together.

Join the event on Triangulate.

#PlayTriangulate #TriangulateLive #OutdoorMultiplayer #LondonGames
```

## Autonomous Agent Model

The safe target is not one uncontrolled bot. It is a small set of agents with clear permissions, rate limits, and approval rules.

### 1. Event Listener Agent

Purpose:

- Watch the public events list.
- Detect content-worthy event changes.
- Create structured content jobs.

Inputs:

- `list_public_events_near`.
- Website events page.
- Manual campaign calendar.

Outputs:

- Content job JSON.
- Event urgency score.
- Recommended channel mix.

Rules:

- Ignore pending, rejected, cancelled, or private events.
- Strip creator/player identifiers.
- Use venue-level copy only.

Example job:

```json
{
  "trigger": "event_t_minus_24h",
  "event_id": "event-id",
  "title": "Golden Hour Match",
  "venue": "Battersea Park",
  "scheduled_for": "2026-04-24T17:45:00+01:00",
  "angle": "Friday evening outdoor multiplayer session",
  "channels": ["tiktok", "instagram_reel", "instagram_story"],
  "privacy_level": "event_only"
}
```

### 2. Creative Director Agent

Purpose:

- Turn content jobs into platform-specific concepts.
- Choose hook, caption, hashtags, and visual direction.

Outputs:

- TikTok script.
- Reel script.
- Story frames.
- Caption and hashtags.
- Asset brief.

Guardrail:

- No individual player narrative.
- No invented turnout.
- No fake live claims.

### 3. Asset Production Agent

Purpose:

- Generate or assemble short videos, stills, and carousels.
- Use brand visuals from the website: glowing triangles, outdoor city/park energy, bold captions.

Asset types:

- 9:16 animated triangle map.
- 9:16 event poster.
- 1:1 Instagram feed graphic.
- 4 to 6 frame carousel.
- Story countdown set.

### 4. Publishing Agent

Purpose:

- Schedule or publish approved content.

Allowed methods:

- Manual upload during Phase 1.
- Official TikTok Content Posting API after developer app approval.
- Official Instagram Platform APIs after Meta setup and app review.
- Approved third-party scheduler only if it uses official platform APIs.

Rules:

- Respect platform rate limits.
- Queue drafts if an API fails.
- Never use unofficial browser automation to bypass API limits.
- Keep a post log with URL, caption, timestamp, event trigger, and asset source.

### 5. Engagement Agent

Purpose:

- Find relevant conversations and suggest replies that reference Triangulate events.

Targets:

- Local gaming nights.
- Social sports events.
- Running clubs and park meetups.
- University societies.
- Freshers and student events.
- Outdoor fitness groups.
- City festivals and community calendars.
- Geocaching, capture-the-flag, park games, and urban exploration communities.

Safe engagement workflow:

- Search for relevant public posts using native platform search, official APIs where allowed, or manual review.
- Draft a reply.
- Score relevance.
- Human approves the reply at first.
- Only move to auto-reply if the platform officially supports the action and account risk is acceptable.

Comment style:

- Helpful, local, and specific.
- Reference Triangulate only when there is a genuine overlap.
- Mention the event, not the people attending it.

Example comments:

```text
This kind of open space is exactly what makes a Triangulate event work: three points, one team shape, real movement. We have a London session coming up on triangulate.live/events.
```

```text
If your group likes outdoor games, Triangulate is built for this: quick to learn, tactical once everyone spreads out. Events are listed at triangulate.live/events.
```

```text
That venue has perfect triangle-game energy: open lanes, good sightlines, lots of room to reset. Would love to see a Triangulate session there.
```

Do not comment:

- On unrelated posts.
- On competitor posts repeatedly.
- On posts involving minors unless there is a formal school/parent/organisation context.
- With the same copy more than once.
- With pressure, fake urgency, or invented claims.

### 6. Safety And Compliance Agent

Purpose:

- Block risky posts before publishing.

Checks:

- Does this mention or identify a player?
- Does it imply someone attended without consent?
- Does it use an image of a real person without consent?
- Does it claim affiliation with another event?
- Does it use a platform action that official APIs do not allow?
- Does it repeat a hashtag/comment pattern too often?
- Does it reveal precise live locations in a way that creates safety concerns?

Default decision:

- Publish event-level promotional content.
- Require human approval for direct interaction with other accounts.
- Reject anything involving personal/player data.

### 7. Analytics Agent

Purpose:

- Measure which formats drive event-page visits and app installs.

Track:

- Post URL.
- Channel.
- Event trigger.
- Hook type.
- Hashtags.
- Views.
- Watch time.
- Saves.
- Shares.
- Profile visits.
- Website clicks.
- Event joins or registrations where attributable.

Weekly report:

- Top 5 posts.
- Best hook.
- Best event trigger.
- Best local hashtags.
- Worst-performing patterns to stop.
- Next week's experiment.

## Platform Notes

TikTok:

- TikTok supports direct posting through the Content Posting API, but apps need approval/audit before public posting is fully available.
- Unapproved/unaudited clients can be restricted to private visibility.
- TikTok also requires user authorization and correct scopes.

Instagram:

- Instagram automation should use Meta's official Instagram APIs and a Professional account.
- Content publishing, comment management, messaging, insights, mentions, and hashtag features require the correct account setup and permissions.
- Broad autonomous liking/commenting on other people's content is not a safe default and should not be done through unofficial automation.

Compliance default:

- Official API or manual only.
- No scraping.
- No browser automation.
- No mass commenting.
- No fake engagement.

- TikTok Content Posting API: https://developers.tiktok.com/products/content-posting-api/
- TikTok Direct Post docs: https://developers.tiktok.com/doc/content-posting-api-reference-direct-post
